Comandos de git

  • git init Sirve para que git empiece a registrar los cambios
  • git status Sirve para ver el estatus de nuestros archivos o cambios
  • git add -A Sirve para agregar los cambios nuevos en nuestro proyecto
  • git commit -m Sirve para poner un comentario de lo que hicimos
  • git log Sirve para mostrar mis commits
  • Google
  • Git branch Te muestra las ramas
  • Git branch nombreDeLaRama Creas una rama
  • Git checkout nombreDeLaRama Me mueve a la rama que yo le indico
  • Git merge nombreDeLaRama Fuciona la rama a tu proyecto principal
  • Git remote add origin linkGitHub Hace la conexion al repositorio de github
  • Git remote -v Verificar la conexion del repositorio
  • Git push origin nombreDeLaRama Envia los archivos al repositorio
  • Git tag Sirve para crear mis versiones
  • Git push origin --tag Sirve para subir mis tags a mi repositorio
  • Git tag -a v1.0.0 -m "" Sirve para crear la descripcion de mi version
  • Git describe --tags $(git rev-list --tags --max-count=3) Sirve para mostrar los ultimos tags
  • Git log > commit.txt Para crear un archivo txt y tener un respaldo con los commits
  • git reset --soft HEAD~1 Sirve para borrar el ultimo commit PERO permite que los cambios en los archivos sigan existiendo
